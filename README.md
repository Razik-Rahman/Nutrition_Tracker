# Nutrition Tracker App

A mobile nutrition tracking application built using Flutter, Dart, and MongoDB.

## Features

* Search food items
* View nutritional information
* Category-based filtering
* Add new food items
* Track calories, protein, fibre, and serving size
* Modern and responsive UI

## Tech Stack

* Flutter
* Dart
* MongoDB


## Getting Started

### Clone the repository

```bash
git clone <your-github-repo-link>
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```
